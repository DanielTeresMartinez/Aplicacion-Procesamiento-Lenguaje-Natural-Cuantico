# Reorganizing Text

## Objective
Reorganize, restructure, and improve the writing of a specific section or subsections in the user's LaTeX document. 

### More information (optional to process)
The topics that could be treaten in the text are: Basic knowledge of quantum computing, Natural Language Processing (NLP) concepts, embeddings, word2Vec model, Quantum word2vec model or explanation of the code developed. 

## Instructions
1. **Analyze the Request**: Identify which section or subsection the user wants to reorganize based on their prompt.
2. **Read the Content**: Read the current text of the specified section. Pay close attention to the logical flow, transitions, clarity of ideas, and overall structure.
3. **Determine the Goal**: The objective is to make the text more coherent, logically structured, and academically appropriate for a Bachelor's Thesis (TFG).
4. **Draft the Restructured Text**: 
    - Reorder paragraphs or subsections to improve the argumentative flow.
    - Enhance the vocabulary, grammar, and phrasing to maintain a formal, academic tone.
    - Ensure smooth and logical transitions between different ideas and paragraphs.
    - Preserve any LaTeX syntax (e.g., `\cite{}`, `\textbf{}`, equations, lists, and references) exactly as they are or adapt them correctly to the new structure.
5. **Language Requirement (CRITICAL)**: The rewritten, improved, and restructured text **MUST** be in **Spanish**, because the thesis is written in Spanish.
6. **Apply the Changes (MANDATORY)**: Once the text has been rewritten, replace the old content (that was given the suggestions to change) with the new one using the appropriate **file editing tools**. Finally, provide the user with a brief summary of what was reorganized and why the new flow is better.
